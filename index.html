Learning to Write Web Pages

How To Create a Web page

Disclaimer: This page was done with the intention of learning how to create a web page.

Web pages are written using html - (Hypertext Markup Language).
Everything in html is contained in tags.

You do not need a fancy editor to create web pages. Any basic text editior should work.
This particular page was written using notepad++. However, you can as easly use your favorite text editor.
For more information see here.

Creating Your HTML File

Fire up your favorite text editior and create a file named index.html. Save the file anywhere you want. How about in ~home/webpage (using Microsoft Windows: Documents\webpage)?
Wondering why the file name must be index.html? It just happens this will be the landing page when someone visits your website. Remember a website can have one or more pages. However, no matter how many pages there are the index page will be the landing page.

HTML Page Structure

Everything on your webpage must be contained in the tags. So, at its most basic starting point a web page will start like this: 

<DOCTYPE html>
<html>

<head>
...header information goes here...
</head>

<body>
...body goes here...
</body>

</html>

Notice the first line of the the document specifies the document as type html.
From here the real meat and bones of the html file begins. It starts with the opening html tag (<html>) and ends with the closing html tag (</html>).
There are two sections:

  1. Head
  2. Body

Page Header

Header content is not shown directly on the web page. Header, at a minimum contains the page title enclosed in title tags (<title></title>).
For simple pages, styling (<style></style>) information will also be located here. For more complicated webpages styling information is located in a seperate CSS file and a link to that file will be located here.
Other information located in the header section includes links to page icon and sripts. For more information see this list at w3schools.

Page Body

All the visible content on your web page is located in the body section (between the opening and closing body tags).
The following are generally used to organize the page body content:

  • Paragraph <p></p> - Starts a new paragraph
  • Heading <h></h> - Creates a heading
    • Headings are ordered from 1 to 6 and size decreases as the number increases. e.g. a h1 tags are <h1></h1>
  • Emphisis <em></em> - Enclosed text appears in italics
  • stong <strong></strong> - Enclosed text appear bold
    • Emphasis and strong tags can be combined, one inside the next, to display bolded and italized text.
  • List - These come in two flavors
    1. Ordered (numbered) list <ol></ol>
    2. Un-ordered list <ul></ul>
    Regardless of list type list items are created using the <li></li> tag.
    See here for more information on lists.
  • Images <img> - also need source and size information
    • Here is the complete tag used to display the image below: <img src="http://s.hswstatic.com/gif/whiskers-sam.jpg" height="200">
  • hyperlink <a> - also need a reference to the external or internal link and the text to display
    • The followign will create a hyperlink to google.com
    <a href="wwww.google.com">google.com</a>

Thats it. A basic guide to web page creation.
