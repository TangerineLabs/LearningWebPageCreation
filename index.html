Learning to Write Web Pages

How To Create a Web page

Disclaimer: This page was done with the intention of learning how to create a web page.

Web pages are written using html - (Hypertext Markup Language).
Everything in html is contained in tags.

You do not need a fancy editor to create web pages. Any basic text editior should work.
This particular page was written using notepad++. However, you can as easly use your favorite text editor.
For more information see here.

Creating your html file

Fire up you favorite text editior and create a file named index.html. Save the file anywhere you want. Just make sure you know where its save location.
Wondering why the file name must be index.html? It just happens

The first line of your html file must specify the the document type as html. This is done using the DOCTYPE! tag like this <DOCTYPE! html>.

Everything on your webpage must be contained in the <html></html> tags. So at its most basic starting point a web page will start like this: 

<DOCTYPE html>
<html>

<head>
...header information goes here...
</head>

<body>
...body goes here...
</body>

</html>

As you can see, web pages have to two sections. They are:

  1. Head (<head>Header contents</head>)
  2. Body (<body>Body contents</body>)
Except for the Title (<title>Page Title</title>) information Head content is not shown on the web page. The title is shown in the browser title area.
For simple pages styling (<style></style>) information will also be located here. For more complicated webpages styling information is located in a seperate CSS file.

Headings

You create a heading with the <h></h>tag. There are six heading types: <h1> to <h6>.
